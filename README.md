Este é um site simples criado apenas para testar a funcionalidade de alternância entre modo claro e modo escuro usando HTML, CSS e JavaScript.
O foco do projeto é experimentar estilos, lógica de troca de tema e uma interface básica para testes.

🛠️ **Tecnologias**
HTML — Estrutura do site
CSS — Estilo dos temas claro e escuro
JavaScript — Controle da lógica de alternância de tema
